# FoodTrailerJsonApp

## Overview

FoodTrailerJsonApp is a simple C# console application that demonstrates how to read and process JSON data. The application loads a JSON file containing an array of crew members and displays their information in the console.

## Features

* Reads data from a JSON file (`crew.json`)
* Deserializes JSON into C# objects
* Iterates through an array of objects
* Displays formatted output in the console

## Technologies Used

* C#
* .NET Console Application
* System.Text.Json

## How to Run

1. Clone the repository:

   ```
   git clone https://github.com/tabner0320/FoodTrailerJsonApp.git
   ```
2. Navigate to the project folder:

   ```
   cd FoodTrailerJsonApp
   ```
3. Run the application:

   ```
   dotnet run
   ```
